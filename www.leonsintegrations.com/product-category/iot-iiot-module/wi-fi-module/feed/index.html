]]> SMWT-1E is a single board solution based on ESP8266 with inbuilt trace antenna. With the complete and self-contained Wi-Fi networking capabilities, SMWT-1E can perform either as a standalone application or as the slave to a host MCU. When SMWT-1E hosts the application, it promptly boots up from the flash. The integrated highspeed cache helps to increase the system performance and optimize the system memory.

Features

  • 802.11b/g/n protocol
  • Wi-fi Direct(P2P), soft-AP
  • Integrated TCP/IP protocol stack
  • Integrated TR switch, balun, LAN, Power amplifier and matching network
  • Integrated PLL, regulators, and power management units
  • Integrated temperature sensor
  • Integrated low power 32-bit CPU
  • Can be serving as a Wi-fi adapter
  • Support single and multi connection mode
  • Range of 100 meters(200-300 meters in free line-of-sight)

 

Applications/Uses

  • Wi-Fi Access point
  • Wi-Fi Station
  • TCP/IP, HTTP Communication with Server
